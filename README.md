### Steps to install

<ul>
    <li>
        Copy and edit config .env file
    </li>
    <li>
        Run command<br>
        <code>php artisan key:generate</code><br>
        <code>php artisan migrate</code><br>
        <code>php artisan db:seed</code><br>
        <code>php artisan optimize:clear</code>
    </li>
</ul>
