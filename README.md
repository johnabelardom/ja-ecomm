### Steps to install

<ul>
    <li>
        Copy and edit config .env file
    </li>
    <li>
        Run command
        <code>php artisan key:generate</code>
        <code>php artisan migrate</code>
        <code>php artisan db:seed</code>
        <code>php artisan optimize:clear</code>
    </li>
</ul>
